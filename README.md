

# rango
 实现各种虚拟银行系统，主要模拟第三方与银行交互过程。


## 结构

数据库：mysql
语言：java
框架：springMVC
页面：Bootstrap

## 开发技术点



### Tools
技术点：
1、数据库池链接与查询，增，改，包括mysql，mongodb
2、nodejs本地命令执行，远程命令执行和交换验证。
3、log4j日志系统。
4、日志通过文件方式访问并，默认显示最后一行。
5、文本文件读写。
6、gbk文件处理。与utf-8互转。
7、报表文件生成与下载。
8、关键字断言与交互。
9、用户，注册，登录，md5加密等。
10、终端日志推送。
11、db2数据库操作和过滤。
